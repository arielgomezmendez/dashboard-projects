#  Project Dashboard (Next.js)

A simple project management dashboard built with **Next.js App Router**, where users can view, filter, and create projects.

This project demonstrates modern frontend practices such as **client/server component separation, dynamic routing, state management, and local persistence**.

---

## Features

- View a list of projects
- Filter projects by status (Active / Completed)
- View project details using dynamic routes
- Create new projects via a form
- Persist data using `localStorage`
- Loading and error states (`loading.tsx` and `error.tsx`)

---

##  Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

---

##  Getting Started

Clone the repository
Install dependencies: npm install

Run the development server: npm run dev
